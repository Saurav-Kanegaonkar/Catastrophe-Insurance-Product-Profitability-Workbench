# Data Notes

All data in this folder is synthetic and generated by `scripts/score_operating_data.py`.

## Files

- `product_segments.csv`: segment-level product economics by county, product program, occupancy type, wildfire tier, market lane, expected loss, premium, capacity, readiness, and recommendation.
- `guideline_scenarios.csv`: modeled guideline and product-change scenarios with premium, loss, margin, combined-ratio, and implementation-risk outputs.
- `implementation_plan.csv`: workstreams required to move recommended product changes into production.
- `product_language_reviews.csv`: policy form, product language, underwriting guideline, and agent practice items that need decision review.

## Synthetic Data Method

The generator uses a fixed seed so the outputs are reproducible. County wildfire tiers drive catastrophe loss ratio, reinsurance load, and capacity pressure. Product programs drive expense load, commission, occupancy mix, premium factors, and implementation complexity. Mitigation credit lowers catastrophe loss and premium. Agent readiness and guideline state affect whether a segment is treated as an advance, watch, or fix-first lane.

The data is designed for interview discussion and portfolio demonstration. It should not be interpreted as real market share, filed pricing, carrier performance, or policyholder data.
